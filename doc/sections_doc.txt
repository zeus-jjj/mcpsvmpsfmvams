Параметры секции start:
-text (обязательный)
--Тип: строка
--Описание: Текст сообщения. Поддерживает markdown и плейсхолдеры ({first_name}, {last_name}, {username}, {crypted_user_id}).

-buttons (необязательный)
--Тип: массив массивов объектов
--Описание: Список кнопок.
--title (обязательный)
---Тип: строка
---Описание: Текст на кнопке.
--callback (необязательный)
---Тип: строка
---Описание: Данные для обработки callback.
--link (необязательный)
---Тип: строка
---Описание: URL ссылка для кнопки.
--web_app (необязательный)
---Тип: строка
---Описание: URL веб-приложения для кнопки.

-action (необязательный)
--Тип: объект
--Описание: Действие при запуске.

-func (обязательный)
--Тип: строка
--Описание: Имя функции.
---channel (необязательный)
---Тип: строка
---Описание: Канал для проверки подписки.
--is_ok (обязательный)
---Тип: строка
---Описание: Метка удачного выполнения действия.

-file (необязательный)
--Тип: объект
--Описание: Параметры для отправки файла.
--content_type (обязательный)
---Тип: строка
---Описание: Тип контента (video, image, document).
--file_path (обязательный)
---Тип: строка
---Описание: Путь к файлу на сервере.
--tg_filename (обязательный)
---Тип: строка
---Описание: Имя файла в Telegram.
--thumbnail (необязательный)
---Тип: строка
---Описание: Путь к файлу превью.

-notifications (необязательный)
--Тип: массив объектов
--Описание: Список отложенных сообщений.
--message (обязательный)
---Тип: строка
---Описание: Метка уведомления.
--at_time (обязательный)
---Тип: объект
---Описание: Параметры времени отправки.
---time (необязательный)
----Тип: строка
----Описание: Время в формате HH:MM.
---delta_days (необязательный)
----Тип: целое число
----Описание: Сдвиг в днях. По умолчанию = 1.
---wait_seconds (необязательный)
----Тип: целое число
----Описание: Время ожидания в секундах. Нельзя использовать совместно с time и delta_days.


Секция callback меет такие-же параметры, как и start-секция, а также:
-delete (необязательный)
--Тип: булевое значение
--Описание: если true, то сообщение на котором сработала кнопка с callback будет удалено
